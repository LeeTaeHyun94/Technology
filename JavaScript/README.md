# 3. Grammar

## 3.1 비교 연산자 (==, ===)

### 3.1.1 동등 연산자 (==)

```다른 언어들과 마찬가지로 좌항과 우항을 비교해서 서로 값(value)이 같다면 true, 다르다면 false를 반환한다.```

### 3.1.2 일치(완전 항등) 연산자 (===)

```
다른 언어들과 다르게 자바스크립트에 존재하는 좌항과 우항이 정확하게 일치할 때 true, 다르다면 false를 반환하는 연산자
정확하게 일치한다는 의미는 값(value)뿐만이 아니라 데이터형까지 엄격하게 일치해야 한다는 것이다.
```
